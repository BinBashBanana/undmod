# Underrated Non-Demons

Underrated non-demons list integration.

## Features

- Levels with inconsistent difficulties now have demonic horns!
- Click on the difficulty face on the level page to learn more.
- Access level lists and map packs through the popup accessible above the treasure room door button!
- All data is provided by the [UND Spreadsheet](https://docs.google.com/spreadsheets/d/1-Abvx7zXRAqpGFVbdTXpn6g1TRYp9WuZqW2pHWE7Dr4/view).
